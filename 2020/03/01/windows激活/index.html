windows激活

2020-03-01
Windows相关
字数统计: 242字   |   阅读时长≈ 1分钟
  1. 管理员权限运行cmd或Windows PowelShell，输入wmic os get caption 查看系统版本
  2. 打开 https://docs.microsoft.com/en-us/previous-versions/windows/it-pro/windows-server-2012-R2-and-2012/jj612867(v=ws.11) 查看自己系统版本对应的激活密钥，以下为部分常用版本密钥
Operating system edition KMS Client Setup Key
Windows 10 Professional W269N-WFGWX-YVC9B-4J6C9-T83GX
Windows 10 Enterprise NPPR9-FWDCX-D2C8J-H872K-2YT43
Windows 10 Education NW6C2-QMPVW-D7KKK-3GKT6-VCFB2
Windows 7 Professional FJ82H-XT6CR-J8D7P-XQJJ2-GPDD4
Windows 7 Enterprise 33PXH-7Y6KF-2VJC9-XBBR8-HVTHH

​ P.S. win 10 专业版激活成功，其他版本未测试

  1. 在cmd中输入 slmgr /ipk 产品密钥 ，我输入的为slmgr /ipk W269N-WFGWX-YVC9B-4J6C9-T83GX

  2. 在cmd中输入 slmgr /skms kms.chinancce.com ，其中kms.chinancce.com为kms服务器地址，kms服务器地址可能失效，以下为部分kms服务器地址，如果都失效了请右转百度or谷歌

  3. 在cmd中输入slmgr /ato ，激活成功
